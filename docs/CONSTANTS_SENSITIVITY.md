This follows the [Tier-2 provenance audit](CONSTANTS_PROVENANCE.md): change one
constant group at a time, measure its effect on the internal perfect-crystal
and uniform-strain checks, and compare the perfect-crystal result to an
independent external calculator.

```bash
python scripts/benchmark_constants_sensitivity.py
```

## Independent external reference

Sergey Stepanov's [X0h / X-Ray Server](https://x-server.gmca.aps.anl.gov/x0h.html)
was queried on 2026-07-18 for:

- GaAs, (004), 10.000 keV
- symmetric Bragg geometry, σ polarization
- room-temperature database structure (`a = 5.6532 Å`)
- all available dispersion databases (X0h/International Tables, Henke,
  Brennan–Cowan, Windt, Chantler)

X0h results:

- Bragg angle: **26.017°**
- σ Darwin FWHM: **5.4238–5.5212 arcsec**
- \(|\operatorname{Re}\chi_h| = 1.0360\times10^{-5}\)** (X0h database)
- off-Bragg absorption depth: **49.14 μm** (X0h database)
- Bragg extinction length: **1.670 μm**

The corresponding semi-infinite perfect-crystal curve from X0h/GID_sl has
peak reflectivity **0.97283**.

This is an external calculation, not a value generated by this repository.
X0h is also reference 58 in Jo et al., Sci. Rep. **12**, 16606 (2022).

## What explains the external difference?

The archival numerical width (6.011″) is **8.9–10.8% broader** than X0h.
Replacing \(a,f_0,f',f''\) does **not** close the gap; without thermal motion,
modern constants make it 1.8% wider.

The dominant missing physical constant is the **300 K Debye–Waller factor**:

- Debye–Waller alone narrows the code curve by 6.85%.
- All audited constants + Debye–Waller give 5.722″, reducing the X0h
  disagreement to **3.6–5.5%**.
- The standard nonabsorbing two-beam formula
  \[
  \Delta\theta_D =
  \frac{2|\chi_h|}{\sin(2\theta_B)},\qquad
  |\chi_h|=\frac{r_e\lambda^2|F_h|}{\pi V_c}
  \]
  gives **5.404″** with all audited constants + Debye–Waller: within
  **0.4–2.1%** of the X0h range.

Thus the audited structure factor is consistent with X0h. The remaining
3.6–5.5% width difference belongs to the numerical/formulation layer
(absorbing finite-slab FWHM versus X0h's semi-infinite Darwin definition,
polarization/boundary conventions, and the fourth-order implementation), not
to a wrong energy or a single bad constant. That is now a focused Tier-1
formula audit item.

## Other robustness findings

- **Peak reflectivity is insensitive to the table changes:** all cases move
  by ≤0.27%. The code gives 0.986 versus X0h/GID_sl 0.9728 (about 1.4% high).
  Modern constants + Debye–Waller do not improve this; it is also a
  formulation/boundary-condition question.
- **Strain-to-angle conversion is exceptionally robust:** the ε=0.002 layer
  shift changes by only 0.09″ (<0.05%) across all cases. The internal
  \(\Delta\theta=-\varepsilon\tan\theta_B\) validation is not sensitive to
  these constants.
- **Absolute Bragg angle is not robust to rounded \(a\):** modern \(a(300 K)\)
  remains required for predictions on an absolute θ axis. Relative
  \(\theta-\theta_B\) plots are unaffected.

## Recommendation

1. **Keep the archival constants as the default for now.** They preserve exact
   notebook/paper lineage, and changing to modern \(f_0,f',f''\) alone does not
   improve agreement with X0h.
2. **Do not treat the omitted Debye–Waller factor as negligible.** A future
   externally benchmarked `gaas_004_10kev_300K` constants/model option should
   include modern \(a,f_0,f',f''\) **and** species-specific 300 K
   Debye–Waller factors as one coherent set.
3. **Complete the Tier-1 formula audit before making that mode production
   default.** Resolve the residual numerical FWHM and reflectivity definitions
   against X0h/GID_sl.
4. For paper Fig. 3 validation, continue to report
   \(\theta-\theta_B\); the current strain-induced asymmetry and peak-shift
   conclusions are unchanged.
